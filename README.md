## Spawtz Tech Screen

A simple multi-tenanted app for displaying the results of social sports competitions

You'll want Visual Studio (express is fine), by default LocalDb is used (same syntax as Sql Server T-SQL) and databases are created and populated automatically

Please take a look at the code, get the app running and see if you can get a feel for how it works - paying particular attention to the tenant Plugin API allowing components to vary by site

We will make changes to this app in the Tech Screen process (instructions provided during that process)

## Spawtz Tech Screen

A simple multi-tenanted app for displaying the results of social sports competitions

You'll want Visual Studio to run it ([Community edition](https://visualstudio.microsoft.com/vs/community/) is fine), by default LocalDb is used (same syntax as Sql Server T-SQL) and databases are created and populated automatically. You can use [SSMS](https://learn.microsoft.com/en-us/ssms/install/install) or any T-SQL-compatible GUI to view/query the databases.

Please take a look at the code, get the app running and see if you can get a feel for how it works - paying particular attention to the tenant Plugin API allowing components to vary by **tenant**

We will make changes to this app in the Tech Screen process (instructions to follow)
